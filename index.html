Catan Board Randomizer

Enter tiles in spiral order. Add harbors after P using only letters/numbers, e.g. P1T3W5O.
Code format: tile tokens in spiral order, then harbor tokens after P. Harbor token format: slot + type (letters/numbers only). Example: P1T3W5O. Harbor types: T=3:1, W=wood, B=brick, S=sheep, G=wheat, O=ore.
